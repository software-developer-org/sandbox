# **Bash Commands**


**creating a file**

|    Syntax        |
|------------------|       
|`touch <filename>`|


**create a folder**

|      Syntax        |
|--------------------|
|`mkdir <foldername>`|


**copy a file**

|            Syntax        |
|--------------------------|
|`cp <filename> <copyname>`|


**rename files/ folders**

|          Syntax        |
|------------------------|
|`mv <oldname> <newname>`|


**delete files**

|    Syntax     |
|---------------|
|`rm <filename>`|
